# Student Journal 📚

Консольная программа для ведения журнала студентов, написанная на языке C.

## Функциональность
- Добавление студентов с ФИО и оценкой
- Просмотр списка всех студентов
- Сохранение данных в текстовый файл
- Загрузка данных из файла при запуске
- Автоматическое ограничение оценки (от 2 до 5)

## Структура проекта
- `Main.c` — главный файл с меню
- `Jurnal.c` — функции для работы со студентами
- `jurnal.h` — заголовочный файл с объявлениями функций

## Как запустить
1. Открой проект в Visual Studio
2. Нажми Ctrl+F5 для запуска

## Технологии
- Язык: C
- IDE: Visual Studio
- Хранение данных: текстовый файл (jurnal.txt)
- ## Скачать
Скачай [Students.exe](Students.exe) и запусти — установка не нужна.
